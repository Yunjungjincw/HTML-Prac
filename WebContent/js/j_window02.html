window 객체


		window.resizeTo(x,y) : 절대좌표로 크기조절 : 절대사이즈
  	window.resizeBy(x,y) : 상대좌표로 크기조절 : 상대사이즈. 기존크기에 대하여 ()안에 수치 만큼 증가
  	
  	
		window.moveTo(x,y) : 절대좌표로 이동 ( 지정된 위치로 이동)
  	window.moveBy(x,y) : 상대좌표로 이동  ( 현대 위치를 기준으로 이동)
  	up:win.moveBy(0,-100);
		down:win.moveBy(0,100);
		left: win.moveBy(-100,0);
		right:win.moveBy(100,0);
			
		
		
		
		★this=> 여기에서는 이벤트가 발생한 객체를 의미.
