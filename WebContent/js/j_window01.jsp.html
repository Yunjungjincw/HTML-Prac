window 객체

  	window 객체는 BOM (Browser Object Model)에서 가장 최상위 객체.
  	window.alert(); 해야지만 생략해서 alert(); 해도 동일한 의미.
  	window.하위객체
  	예)window.location
  	window.history
  	window.document
  	window.document.forms
  	window.document.forms[].text
  	
  	--------
  	js에서 정수로 변환 
  	parseInt(string) : 문자열을 정수의 Number타입으로 변환 
  	parseFloat(string) : 문자열을 실수의 Number타입으로 변환
