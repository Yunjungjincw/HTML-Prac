The display Property

선택자{ display inline|inline-block|block 등}
inline 레벨요소 : width, height 적용 x
한 줄에 여러 요소가 나열
예> span, img, a 등

inline-block 레벨 요소 : width, height 적용 o
한 줄에 여러 요소가 나열

block 레벨 요소 : width, height 적용 o
한 줄에 단 1개 요소가 차지
예) div, hr, h1~h6, p, li 등
